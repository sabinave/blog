Scope the project

Posted by Evelina Hedberg on Monday May 6, 2019
We want to develop a great project with different extension depending on our resources and our time.

Extensions

- We want to try to add density to the different liquids in order to switch order of the fluids within the filled object. To do this in a satisfying way we probably require some soft implementation of SPH( Smoothed Particle Hydrodynamics) in order to simulate the fluids interacting with each other when switching places.

- We want to add a function where the liquids are mixed in a realistic way depending on the different densities.

- We want to use above implementations to create a user-friendly component where one could use the tool to try out mixing different liquids and shaking them in an interactive way.
